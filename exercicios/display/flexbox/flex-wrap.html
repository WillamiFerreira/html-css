* E shorthand flow-flex

Eles irão encolher mesmo se a largura dada aos itens for em px.

Shorthand Flex-flow ✋

Podemos aplicar duas propriedades( flex-diction e flex-wrap) de uma vez só usando uma shorthand, a flex-flow.

Em vez de usarmos esse códigos com as duas propriedades separadas...

.flex-counteiner{ flex-direction:row; flex-wrap: nowrap; }

Podemos usar a shorthand Flex-flow que o resulado será o mesmo.

.flex-counteiner{ flex-flow:row nowrap; }

o flex-content vem antes!!
